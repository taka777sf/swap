## GitHub Pagesへの公開・更新手順

1. リポジトリのCodeタブを開く
2. 「Add file → Upload files」（または既存ファイルを開いて中身を書き換え）で、
   このフォルダの中身（index.html, style.css, app.js, manifest.json, service-worker.js,
   icon-192.png, icon-512.png, apple-touch-icon.png）を**すべてルート直下に**アップロード
   - 同じファイル名なら上書きされます。フォルダ分けは不要です
3. Settings → Pages で Branch=main / root になっていることを確認（初回のみ）
4. 数分待って `https://taka777sf.github.io/swap/` にアクセス

## iPhoneでアプリ化する

1. iPhoneのSafariで上記URLを開く
2. 共有ボタン→「ホーム画面に追加」
3. 既にアプリを追加済みの場合は、一度アプリを開いて数秒待つ（Service Workerが自動更新）

## データとバックアップについて

- 入力データはこの端末（iPhone）のローカルストレージに保存されます。他の端末とは自動同期されません
- 永続化リクエストにより消去されにくくなっていますが、100%の保証はありません
- ヘッダーの「⇩ 保存」で定期的にJSONバックアップを書き出し、iCloud DriveやPC等に保管することを推奨します
- 「⇧ 復元」で直近のバックアップファイルを読み込めば元に戻せます
- 複数端末での同期や、友達とのデータ共有が必要になったら、サーバー保存（VPS等）への切り替えも可能です
